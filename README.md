# terraVE
TerraVE es una plataforma web que combina memoria ciudadana con visión computacional para responder dos preguntas críticas post-desastre: ¿Qué existía aquí? → Levantamiento ciudadano de infraestructura perdida ¿Qué quedó en pie? → Clasificación automática de daño estructural por foto El resultado: un mapa vivo.
